# Manual de Usuario

## Dashboard Ejecutivo

Permite:

* Consultar calorías por restaurante
* Comparar cadenas de comida rápida
* Revisar productos más saludables
* Consultar alimentos externos mediante USDA API

## Dashboard Técnico

Permite:

* Revisar calidad de datos
* Analizar valores nulos
* Revisar distribuciones estadísticas
* Analizar correlaciones nutricionales
