# Guía de Instalación

## Clonar repositorio

git clone URL_REPOSITORIO

## Crear entorno virtual

python -m venv venv

## Activar entorno

Windows:
venv\Scripts\activate

Linux:
source venv/bin/activate

## Instalar dependencias

pip install -r requirements.txt

## Ejecutar Dashboard

streamlit run dashboards/ejecutivo.py
